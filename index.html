Hatch and Match!

The aim of this game is to uncover all the pairs of matching birds.

First choose any card. Then choose a second to see if you can find a matching bird!

If the cards do not match they will be turned face dowen again.

Use your memory to remember where the birds are so you can finish the game in the least number of card flips possible!
